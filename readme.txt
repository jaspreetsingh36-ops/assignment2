# Portfolio Website - Assignment 2

A professional personal portfolio website built with **Express.js**, **EJS templates**, and **Bootstrap**. This project demonstrates modern web development skills including server-side rendering, responsive design, and form validation.

##  Live Demo

**Live Website:** [https://assignment2-3ero.onrender.com/](https://assignment2-3ero.onrender.com/)

##  Project Overview

This portfolio website showcases my skills, projects, and professional information as a Networking & IT Security student. The site features a clean, responsive design with four main pages and advanced form validation.

##  Technologies Used

- **Backend**: Node.js, Express.js
- **Templating**: EJS (Embedded JavaScript)
- **Frontend**: Bootstrap 5, Font Awesome Icons
- **Styling**: Custom CSS with gradient designs
- **Deployment**: Render (Cloud Platform)

##  Project Structure

```
assignment2/
├── public/
│   ├── images/          # Profile images and assets
│   ├── stylesheets/     # Custom CSS styles
│   └── javascripts/     # Client-side JavaScript
├── views/
│   ├── partials/        # Reusable components
│   │   ├── header.ejs   # Navigation with profile image
│   │   └── footer.ejs   # Footer with social links
│   ├── index.ejs        # Home page
│   ├── about.ejs        # About me page
│   ├── projects.ejs     # Projects showcase
│   ├── contact.ejs      # Contact form with validation
│   └── layout.ejs       # Main layout template
├── routes/
│   └── index.js         # Express routes for all pages
├── app.js               # Main Express application
├── package.json         # Project dependencies
└── Procfile             # Deployment configuration
```

##  Installation & Setup

### Prerequisites
- Node.js (v14 or higher)
- npm package manager

### Local Development
1. Clone the repository:
```bash
git clone https://github.com/jaspreetsingh36-ops/assignment2.git
cd assignment2
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open your browser and visit:
```
http://localhost:3000
```
